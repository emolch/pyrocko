
.. program:: pyrocko

Pyrocko - *package introspection*
=================================

The :program:`pyrocko` command line tool provides information about the Pyrocko
installation in use.

Command reference
-----------------

The :app:`pyrocko` tool and its subcommands are self-documenting with the
``--help`` option. Run ``pyrocko`` without any options to get the list of
available subcommands.  Run ``pyrocko SUBCOMMAND --help`` to get details about
a specific subcommand, e.g. ``pyrocko info --help``.
